# 🏦 Bank Logos - SVG Files

Esta pasta contém os logos dos bancos em formato SVG.

## 📋 Arquivos Necessários (Prioridade ALTA):

Adicione os seguintes arquivos SVG nesta pasta:

1. ✅ **itau.svg** - Banco Itaú (laranja #EC7000)
2. ✅ **bradesco.svg** - Banco Bradesco (vermelho #CC092F)
3. ✅ **santander.svg** - Banco Santander (vermelho #EC0000)
4. ✅ **bb.svg** - Banco do Brasil (amarelo #FFED00)
5. ✅ **caixa.svg** - Caixa Econômica (azul #0066B3)
6. ✅ **inter.svg** - Banco Inter (laranja #FF7A00)
7. ✅ **c6.svg** - C6 Bank (preto #000000)

## 📐 Especificações dos SVGs:

### Formato:
- **Tipo:** SVG (Scalable Vector Graphics)
- **ViewBox:** `0 0 24 24` ou `0 0 32 32`
- **Cores:** Originais do banco (ou `currentColor` para monocromático)

### Exemplo de SVG válido:
```svg
<svg viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">
  <path d="M12 2L2 7v10c0 5.55 3.84 10.74 9 12 5.16-1.26 9-6.45 9-12V7l-10-5z" fill="#EC7000"/>
</svg>
```

### Otimização:
- ❌ Remover `id`, `class`, `style` inline
- ❌ Remover comentários e metadados
- ✅ Manter apenas elementos gráficos (`path`, `circle`, `rect`, etc.)

## 🎯 Onde Encontrar:

1. **Sites oficiais dos bancos** (seção Imprensa/Mídia)
2. **Simple Icons:** https://simpleicons.org/
3. **SVG Repo:** https://www.svgrepo.com/
4. **Vectorizer.ai:** https://vectorizer.ai/ (converter PNG para SVG)

## 📝 Nomenclatura:

- Use lowercase
- Use hífen para separar palavras
- Exemplos: `itau.svg`, `banco-do-brasil.svg`, `c6-bank.svg`

## ✅ Após Adicionar os SVGs:

O sistema reconhecerá automaticamente os logos e os exibirá nos cards de contas e transações!
