# Vercel environment variables

Configure estas variaveis na Vercel para o projeto web:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `VITE_VAPID_PUBLIC_KEY`

Valores e fluxo completo: [docs/SUPABASE_SETUP.md](/d:/2026/personal-finance/personal-finance-la/docs/SUPABASE_SETUP.md).

## Ambientes

Marque as variaveis em:

- `Production`
- `Preview`
- `Development`
