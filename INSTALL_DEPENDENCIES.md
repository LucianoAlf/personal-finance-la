# 📦 Dependências para Gamificação

Execute este comando no terminal:

```bash
npm install canvas-confetti react-hot-toast
```

## O que cada biblioteca faz:

- **canvas-confetti**: Animações de confete para celebrar conquistas 🎉
- **react-hot-toast**: Notificações elegantes para micro-conquistas 🔔

Após instalar, o sistema de gamificação estará completo!
